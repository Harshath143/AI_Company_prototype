# Technology Stack
We will use Python as our primary programming language, with a focus on object-oriented design. For the frontend, we will use HTML, CSS, and JavaScript.

# Folder Layout
- src/      : main source code files
  - src/snake.py - Snake class, movement logic
  - src/food.py  - Food class, random placement
  - src/game.py  - Game loop, scoring, input handling
- tests/    : unit test files
- frontend/ : HTML/CSS/JS files
  - frontend/index.html - Game canvas and UI
- logs/     : runtime log files
