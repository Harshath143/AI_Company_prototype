# Product Requirements Document: Snake Game
## Introduction
The goal of this project is to build a classic snake game where the player controls a snake that can move up, down, left, or right to eat food pellets and avoid obstacles.
## Functional Requirements
* The game shall have a start screen with options to start a new game, view high scores, and exit the game.
* The game shall have a game board with a size of 20x20 cells.
* The snake shall be represented by a series of connected cells, with the head of the snake being the leading cell.
* The snake shall be able to move up, down, left, or right using the corresponding arrow keys.
* The game shall generate food pellets at random locations on the game board.
* When the snake eats a food pellet, it shall grow by one cell.
* The game shall end when the snake collides with the edge of the game board or with its own body.
* The game shall keep track of the player's score, which shall be the number of food pellets eaten.
* The game shall have a high score list that stores the top 10 highest scores.
## Non-Functional Requirements
* The game shall have a responsive and intuitive user interface.
* The game shall have a frame rate of at least 10 frames per second.
* The game shall be able to run on a variety of devices, including desktop computers, laptops, and mobile devices.
## User Interface Requirements
* The game shall have a start screen with a button to start a new game.
* The game shall have a game board that displays the snake, food pellets, and obstacles.
* The game shall have a score display that shows the player's current score.
* The game shall have a high score list that displays the top 10 highest scores.
## Testing Requirements
* The game shall be tested on a variety of devices to ensure compatibility.
* The game shall be tested to ensure that the snake moves correctly and that the game ends when the snake collides with the edge of the game board or with its own body.
* The game shall be tested to ensure that the score is displayed correctly and that the high score list is updated correctly.
