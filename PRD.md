# Product Requirements Document
## Introduction
The goal of this project is to build a snake game where the player controls a snake that can move up, down, left, or right to eat food pellets. The game ends when the snake runs into the wall or itself.
## Functional Requirements
* The game shall have a grid-based system where the snake and food pellets are placed.
* The snake shall be able to move up, down, left, or right using keyboard inputs.
* The game shall generate food pellets at random locations on the grid.
* The snake shall grow in length when it eats a food pellet.
* The game shall end when the snake runs into the wall or itself.
## Non-Functional Requirements
* The game shall have a user-friendly interface.
* The game shall be able to run on a variety of devices, including desktops and laptops.
